# Notepad-WinForms


Улучшенный текстовый редактор на WinForms с полезными функциями для комфортного написания текста и кода.

## Функционал

**Базовые возможности:**
* Создание, открытие и сохранение текстовых файлов (.txt).

**Расширенные функции:**
* **Счётчик символов, слов и строк** — отображается в статус‑баре внизу окна.
* **Тёмная тема** — переключение между светлой и тёмной темой интерфейса.
* **Автоматическое закрытие скобок и кавычек** — при вводе открывающей скобки/кавычки автоматически добавляется закрывающая.
* **Подсветка текущей строки** — строка с курсором подсвечивается для удобства чтения.
* **Горячие клавиши** — полная поддержка сочетаний:
    * `Ctrl+S` — сохранить текущий файл;
    * `Ctrl+N` — создать новый документ;
    * `Ctrl+O` — открыть существующий файл;
    * `Ctrl+Shift+S` — сохранить как… (выбор имени и пути).

## Технологии
* C#
* Windows Forms (.NET Framework / .NET)
* System.IO (работа с файлами)
* Custom UI styling (тёмная тема)
